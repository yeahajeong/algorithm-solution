### ✔ BFS로 푸는 문제
BFS를 이용하려면 큐를 사용해야한다. \
빠른 성능을 위해 deque (데크 : 양방향에서 데이터를 처리할 수 있는 queue형 자료구조)를 이용!

pop(0) -> O(n) \
popleft() -> O(1)

### ✔ 이중리스트
```python
answer = [[0] * n for _ in range(n)]
answer[0][0] = 1
print(answer) # -> [[1, 0, 0], [0, 0, 0], [0, 0, 0]]

# answer = [[0] * n] * n 이 경우는 돌아가지않는다
# 왜냐하면 안쪽에 적혀있는 [0] * n 이 부분이 모두 같은 리스트로 인식이 되기 때문
answer = [[0] * n] * n
answer[0][0] = 1
print(answer) # -> [[1, 0, 0], [1, 0, 0], [1, 0, 0]]
```
